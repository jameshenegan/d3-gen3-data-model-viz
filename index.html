Metadata Visualization for a Graph-Based Data Model

This tool can be used to visualize a proposed graphical structure for a collection of variables.

  1. The orange circles represent nodes in the graph.
  2. The green circles represent variables in the data dictionary.
  3. Use the mouse-wheel to scroll in and out.
  4. Hover over a circle to see a tooltip showing the name of that variable/node.

If you don't have a mouse-wheel, you can use the buttons below to scroll in and out.

This is similar to the "Dictionary Visualizer" used in Gen3. However, this visualizer lets you see the number of variables in a given node.
